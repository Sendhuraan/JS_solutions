## Generate continuous combinations in an array

Given an array of size n, generate and print all possible continuous combinations of elements in array.

For example, if 
input array is {1, 2, 3, 4}

then output should be
{1},
{2},
{3},
{4},
{1, 2},
{2, 3},
{3, 4},
{1, 2, 3}
{2, 3, 4},
{1, 2, 3, 4}
